# PROPOSAL PROYEK: SISTEM MANAJEMEN PEMBAYARAN IURAN SAMPAH

## 1. Judul Proyek
**APLIKASI MANAJEMEN PEMBAYARAN IURAN SAMPAH (AMPIS)**

## 2. Latar Belakang
Pengelolaan iuran kebersihan atau sampah di tingkat lingkungan (RT/RW) atau perumahan seringkali masih dilakukan secara manual menggunakan buku besar. Hal ini menimbulkan beberapa kendala, antara lain:
- Risiko data hilang atau rusak.
- Kesulitan dalam melacak warga yang belum membayar (tunggakan).
- Proses rekapitulasi laporan pendapatan bulanan yang memakan waktu.
- Kurangnya transparansi dalam pengelolaan dana.

## 3. Tujuan Proyek
Proyek ini bertujuan untuk membangun sebuah sistem berbasis desktop yang modern dan efisien untuk:
- Mendigitalisasi pencatatan pembayaran iuran sampah.
- Memberikan kemudahan bagi pengurus dalam memantau status pembayaran warga (Lunas/Belum Lunas).
- Mengotomatiskan laporan penghasilan bulanan secara akurat.
- Meningkatkan profesionalisme operasional lingkungan melalui antarmuka (UI) yang premium.

## 4. Fitur Utama
Sistem ini dilengkapi dengan berbagai fitur fungsional:
1.  **Keamanan Login**: Akses terbatas menggunakan sistem PIN keamanan.
2.  **Kelola Data Pembayaran**: Pencatatan data pembayaran warga yang mencakup nama, alamat, bulan, dan nominal.
3.  **Manajemen Pelanggan**: Daftar warga dengan status langganan (Aktif/Tidak Aktif).
4.  **Laporan Penghasilan**: Dashboard otomatis yang menampilkan total pendapatan dari bulan ke bulan.
5.  **Filter Cerdas**: Kemudahan mencari data berdasarkan nama dan memfilter record berdasarkan bulan.
6.  **Antarmuka Premium**: Desain modern menggunakan skema warna *Midnight Teal* & *Gold* yang elegan.

## 5. Spesifikasi Teknis (Tech Stack)
Aplikasi dikembangkan menggunakan teknologi terkini:
- **Bahasa Pemrograman**: Python 3
- **Framework UI**: PyQt6 (Desktop GUI)
- **Database**: SQLite (Penyimpanan data lokal yang handal)
- **Styling**: QSS (Qt Style Sheets) untuk kustomisasi UI tingkat lanjut.
- **Packaging**: PyInstaller (Distribusi dalam bentuk file .EXE)

## 6. Manfaat bagi Pengguna
- **Efisiensi Waktu**: Pencarian dan input data jauh lebih cepat dibandingkan metode manual.
- **Akurasi Data**: Meminimalisir kesalahan perhitungan pendapatan.
- **Portabilitas**: Aplikasi dapat langsung dijalankan tanpa perlu instalasi rumit (Standalone EXE).
- **Visualisasi Jelas**: Laporan penghasilan memudahkan perencanaan anggaran kebersihan lingkungan.

## 7. Status Pengembangan
Saat ini aplikasi telah menyelesaikan tahap pengembangan inti, mencakup fungsionalitas CRUD (Create, Read, Update, Delete), sistem filter bulanan, dashboard pendapatan, dan optimasi antarmuka pengguna.
